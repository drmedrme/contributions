Employee self photo Update

With this module, Employee can change photo of them on employee profile. There will be button for capture of image using webcam, By clicking capture button it will open window.

Capture Snap

1) Click on Take snapshot once your webcam started. 2) Click on save photo to see a sample image on right side. 3) Click on save button to save changes. 4) Click on Capture image button to set snapshot on employee profile.

Set Picture

Set captured snapshot to Employee's profile photo.
